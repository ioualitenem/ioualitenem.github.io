Personal Projects

I am currently working on a paper summarizing & simplifying the Black-Scholes-Merton valuation formula. It is a paper aimed at helping people use and/or code the formula while getting the gist of its assumptions, derivations, limitations and extensions.

Concurrently, I am working on an efficient R package about option pricing using sophisticated methods that can be found in mathematical finance litterature.

Please note that some libraries/packages that can "translate" code from one language to another exist; such as reticulate which does Python to R; they may not work the way it was intended to work and should not be relied upon blindly.
